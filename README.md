## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-gradient: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    --accent-color: #667eea;
    /* ... */
}
```

### Form Fields
Add or remove form fields in `index.html` and update validation in `script.js`.

### API Integration
Replace the `simulateSignup` function in `script.js` with your actual API endpoint.
